Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2: Difference Between innerHTML, innerText, and textContent is innerHTML use when you need to insert or retrieve HTML content. Use innerText when you need only visible text (ignores hidden elements). And use textContent when you need all text, including hidden text, and better performance.

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a technique in the DOM where a parent element listens for events on its child elements using event bubbling. Instead of attaching event listeners to multiple child elements, you attach a single listener to the parent and handle events using event.target.

Question-4: What is event bubbling in the DOM?

Answer-4: Event bubbling is a phase in the DOM event propagation where an event starts from the target element and moves upward to its parent elements until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer-5: We can create, add, and remove elements using JavaScript:

Create an Element:

let createElement = document.createElement("div");
createElement.textContent = "Hello, World!";

Add an Element:

document.body.appendChild(createElement);

Remove an Element:

let elementToRemove = document.getElementById("myElement");
elementToRemove.remove();
